Changelog — リリース履歴

App Store 公開済みバージョンの主要変更点をまとめています。 日々の開発トピックやベータ版の試行錯誤は 公式開発日誌 をご覧ください。

表記について

このページは「ユーザーが知っておくと得をする変更」だけを抜粋しています。 内部リファクタリングや軽微な文言修正は除外し、機能追加・体感に効く改善・互換性に影響する変更を中心に記載しています。

v1.x — 2026

v1.0.0 — 2026-02-15（初回リリース）

  • 「メモ→自分宛メール」のコア機能を実装
  • 起動 0.3 秒・送信 150ms を達成（iPhone 13 / iOS 17 計測値）
  • Outbox（オフライン未送信キュー）を標準搭載
  • 送信履歴を端末内に保持
  • Outbox・履歴を AES-GCM (256-bit) で端末内暗号化
  • Cloudflare Workers ベースのエッジ送信基盤
  • Gmail / iCloud / Outlook に対応
  • 7 日間の無料トライアルを提供

初回リリースの背景は about ページ と 「Captioが使えない理由」 を併せてご覧ください。

v1.1 — 計画中

  • 件名プレフィックスのテンプレ機能（memo:, idea:, todo: 切替）
  • 送信履歴の検索 / 絞り込みを高速化
  • iPad レイアウトの最適化
  • ローカライズ追加候補: ko / es / pt-BR / id（公式サイトに準拠）

予定は変更される可能性があります。最新のロードマップ感は 開発日誌 と X (@simplememofast) でも発信しています。
