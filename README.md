# simplememofast.github.io

[Captio式シンプルメモ](https://simplememofast.com/) の Developer Hub（GitHub Pages）。

公式プロダクトサイト [simplememofast.com](https://simplememofast.com/) の補完として、
開発者向けの技術解説、Captio 移行ガイド、リリース履歴、プレスキットなどを公開する静的サイトです。

## サイト構成

| Path | 内容 |
| --- | --- |
| `/` (`index.html`) | Developer Hub トップ。各セクションへの導線。 |
| `/about.html` | 開発元（株式会社ユリカ）、開発者、`rel="me"` 本人確認チェーン。 |
| `/features.html` | Outbox / AES-GCM / Cloudflare Workers / iOS 起動最適化の技術解説。 |
| `/captio-migration.html` | Captio ユーザー向けの移行ガイド、機能対応表、Shortcuts レシピ、FAQ。 |
| `/changelog.html` | App Store 公開バージョンの主要変更点。 |
| `/press.html` | メディア向けプレスキット、boilerplate、ファクトシート、画像素材。 |
| `/sitemap.xml` | 全ページのサイトマップ。 |
| `/robots.txt` | クロール許可 + サイトマップ参照。 |
| `/404.html` | 404 ページ。 |
| `/assets/css/style.css` | 共有スタイル。 |

## デプロイ

リポジトリ名は **必ず `simplememofast.github.io`**（GitHub アカウント名と同一）にしてください。
そうすると `https://simplememofast.github.io/` で User Pages として公開されます。

詳しい手順は [`DEPLOY.md`](./DEPLOY.md) を参照。

## ライセンス・利用

本リポジトリの HTML / CSS は社内ツールとしての配布物です。
画像素材（公式サイト経由でリンクしているもの）の利用条件は [プレスキット](./press.html) を参照してください。
